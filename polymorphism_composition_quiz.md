Polymorphism
What does the word 'polymorphism' mean?

    You can perform a single action in multiple ways.

What does it mean when we apply polymorphism to OO design? Give a simple Java example.

    An example would be creating a sound method for various animals.
    Each animal uses the same sound method, but makes a different sound.

What can we use to implement polymorphism in Java?

    Interfaces or Abstract Classes

How many 'forms' can an object take when using polymorphism?

    Multiple!

Give an example of when you could use polymorphism.

    If you wanted to have an array of different types of classes that implement the same interface.

Composition
What do we mean by 'composition' in reference to object-oriented programming?

    A class that references one or more objects of other classes in instance variables.

When would you use composition? Provide a simple example in Java.

    For example a car has an engine, wheels etc.

What is/are the advantage(s) of using composition?

    Re-use existing code.

When an object is destroyed, what happens to all the objects it is composed of?

    They are also destroyed.
